Detailed Description

Formats and writes solver results to file and/or console.

All output methods are static — this is a utility class with no state.

const ProblemData &  data,
bool  includeRisk = true 
)
inlinestatic

Writes the complete assignment output to a file.

Generates the output file in the format specified by the project PDF. Includes assignment tables (by submission and by reviewer), totals, unfulfilled submissions, and — when includeRisk is true and risk analysis was configured — the risk analysis section as well.

Parameters
filenamePath to the output file.
resultThe solver result to format.
dataThe problem data (for config access).
includeRiskIf true (default) and risk is configured, appends the risk section to the same file. Pass false when the caller intends to write risk to a separate file (batch mode with an explicit risk output path).

where A = assignments, U = unfulfilled, R = risky reviewers (due to sorting).
