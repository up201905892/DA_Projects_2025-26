Detailed Description

Describes a submission that could not receive enough reviews.

When the max-flow is less than n * MinRev, some submissions are under-reviewed. This struct captures which submission, which domain is missing, and how many additional reviews are needed.

Output format (Figure 3 of the PDF):
#SubmissionId, Domain, MissingReviews
31, 3, 2
